# F12 功能键连发 function key presser
给当前窗口设置自动连发按键(功能键F1到F12).  

# 说明
按修改键(默认为右边的Ctrl键)加任意功能键(F1-F12)来启动自动按键.  
启动以后默认每秒自动按一次对应的功能键.  
按修改键(默认为右边的Ctrl键)加退格键(Back)关闭当前窗口所有启动的自动按键.  
按修改键(默认为右边的Ctrl键)加\键(回车上方)来读取修改之后的配置文件(f12.txt).  

# 配置文件
f12.txt为配置文件,可以在f12.txt文件里添加比如

F2 2

来把F2的发送时间改成2秒一次, 或者

F2 2-4

把F2的发送时间改成2-4秒之间的随机间隔
